# Subtractor

## Overview

This folder contains the **transistor-level implementation of CMOS subtractor circuits** designed using **Cadence Virtuoso** in **45 nm CMOS technology**. The collection demonstrates the hierarchical development of arithmetic circuits, beginning with the **Half Subtractor**, progressing to a **1-Bit Full Subtractor** implemented using two Half Subtractors, and extending to **4-Bit Ripple Borrow Subtractors** with and without overflow detection.

These implementations showcase modular digital design principles, hierarchical circuit development, and arithmetic verification through transient simulations.

## Design Details

### Half Subtractor

The Half Subtractor is the fundamental arithmetic circuit used to subtract one binary digit from another. It generates a **Difference** and a **Borrow** output without requiring a borrow input from a previous stage.

#### Logic Equations

```text
Difference = A ⊕ B
Borrow = A' · B
```

**Available Files**

- **Schematic** – Transistor-level CMOS Half Subtractor implementation.
- **Symbol** – Reusable symbol for hierarchical circuit design.
- **Test** – Testbench developed for functional verification.
- **Waveform** – Transient simulation validating Difference and Borrow outputs.

---

### 1-Bit Full Subtractor

The 1-Bit Full Subtractor is implemented **hierarchically using two Half Subtractors**, demonstrating modular and reusable circuit design principles.

It subtracts two binary inputs along with an input borrow (**Bin**) to generate the **Difference** and **Borrow Out**.

#### Logic Equations

```text
Difference = A ⊕ B ⊕ Bin

BorrowOut = A'B + Bin(A ⊕ B)'
```

**Available Files**

- **Schematic** – Hierarchical transistor-level Full Subtractor implementation.
- **Symbol** – Custom symbol for higher-level integration.
- **Test** – Testbench validating all input combinations.
- **Waveform** – Transient simulation confirming correct Full Subtractor functionality.

---

### 4-Bit Ripple Borrow Subtractor

The 4-Bit Ripple Borrow Subtractor is constructed by cascading four 1-Bit Full Subtractors to perform binary subtraction on two 4-bit operands.

```text
A = A₃A₂A₁A₀
B = B₃B₂B₁B₀
```

The borrow generated by each stage propagates to the next higher-order stage, implementing the Ripple Borrow architecture.

**Available Files**

- **Schematic** – Complete transistor-level 4-bit subtractor.
- **Symbol** – Hierarchical symbol for reusable integration.
- **Test** – Testbench covering multiple subtraction cases.
- **Waveform** – Simulation verifying correct binary subtraction.

---

### 4-Bit Subtractor with Overflow Detection

This implementation extends the Ripple Borrow Subtractor by integrating **overflow detection**, enabling reliable signed binary subtraction.

Overflow is detected by comparing the carry into and carry out of the most significant bit (MSB).

#### Overflow Equation

```text
Overflow = Carry_in(MSB) ⊕ Carry_out(MSB)
```

**Available Files**

- **Schematic** – 4-bit subtractor with overflow detection circuitry.
- **Symbol** – Hierarchical reusable symbol.
- **Test** – Testbench including overflow scenarios.
- **Waveform** – Simulation validating both subtraction operation and overflow detection.

## Design Flow

1. Designed the Half Subtractor using CMOS transistor-level logic.
2. Constructed the 1-Bit Full Subtractor using two Half Subtractors.
3. Cascaded four Full Subtractors to implement the 4-Bit Ripple Borrow Subtractor.
4. Integrated overflow detection logic into the 4-Bit Subtractor.
5. Created reusable schematic symbols for hierarchical circuit design.
6. Developed dedicated testbenches for functional verification.
7. Performed transient simulations to validate subtraction operations and overflow detection.
