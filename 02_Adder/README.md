# Adder

## Overview

This folder contains the **transistor-level implementation of CMOS adder circuits** designed using **Cadence Virtuoso** in **45 nm CMOS technology**. The collection demonstrates the hierarchical development of arithmetic circuits, beginning with the **Half Adder**, progressing to a **1-Bit Full Adder** implemented using two Half Adders, and extending to **4-Bit Ripple Carry Adders** with and without overflow detection.

These implementations showcase modular digital design principles, hierarchical circuit development, and arithmetic verification through transient simulations.

## Design Details

### Half Adder

The Half Adder is the fundamental arithmetic building block capable of adding two single-bit binary inputs. It generates a **Sum** and a **Carry** output without considering an input carry.

#### Logic Equations

```
Sum = A ⊕ B
Carry = A · B
```

**Available Files**

- **Schematic** – Transistor-level CMOS Half Adder implementation.
- **Symbol** – Reusable symbol for hierarchical circuit design.
- **Test** – Testbench for functional verification.
- **Waveform** – Transient simulation validating Sum and Carry outputs.

---

### 1-Bit Full Adder

The 1-Bit Full Adder is implemented **hierarchically using two Half Adders**, demonstrating modular and reusable circuit design.

It adds two binary inputs along with an input carry (**Cin**) to generate a **Sum** and **Carry Out**.

#### Logic Equations

```
Sum = A ⊕ B ⊕ Cin

CarryOut = AB + Cin(A ⊕ B)
```

**Available Files**

- **Schematic** – Hierarchical transistor-level Full Adder implementation.
- **Symbol** – Custom symbol for higher-level integration.
- **Test** – Testbench validating all input combinations.
- **Waveform** – Transient simulation confirming correct Full Adder functionality.

---

### 4-Bit Ripple Carry Adder

The 4-Bit Ripple Carry Adder is constructed by cascading four 1-Bit Full Adders to perform binary addition on two 4-bit operands.

```
A = A₃A₂A₁A₀
B = B₃B₂B₁B₀
```

The carry generated by each Full Adder propagates to the next stage, implementing the Ripple Carry architecture.

**Available Files**

- **Schematic** – Complete transistor-level 4-bit adder.
- **Symbol** – Hierarchical symbol for reusable integration.
- **Test** – Testbench covering multiple addition cases.
- **Waveform** – Simulation verifying correct binary addition.

---

### 4-Bit Adder with Overflow Detection

This design extends the Ripple Carry Adder by integrating **overflow detection**, allowing reliable signed arithmetic operations.

Overflow is detected by comparing the carry into and carry out of the most significant bit.

#### Overflow Equation

```
Overflow = Carry_in(MSB) ⊕ Carry_out(MSB)
```

**Available Files**

- **Schematic** – 4-bit adder with overflow detection circuitry.
- **Symbol** – Hierarchical reusable symbol.
- **Test** – Testbench including overflow scenarios.
- **Waveform** – Simulation validating both arithmetic operation and overflow detection.

## Design Flow

1. Designed the Half Adder using CMOS transistor-level logic.
2. Constructed the 1-Bit Full Adder using two Half Adders.
3. Cascaded four Full Adders to implement the 4-Bit Ripple Carry Adder.
4. Integrated overflow detection logic into the 4-Bit Adder.
5. Created reusable schematic symbols for hierarchical circuit design.
6. Developed dedicated testbenches for functional verification.
7. Performed transient simulations to validate arithmetic operations and overflow detection.
